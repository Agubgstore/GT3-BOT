##  Editing the file
Edit the required value in `gtpsconfig.json`.
```json
{
    "exegtps": "server.exe",
    "player": "players",
    "world": "worlds",
    "nameserver": "GTPS"
}

```
- Edit menu display in `display.js`
- Edit owner number etc in `settings.json`

## Run

```bash
npm start
```
